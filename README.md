# gypsum

石膏自定义

![预览图](./imgs/preview.png)

## 用法

警告：目前版本仅供测试，无鉴权，无输入验证，禁止用于生产环境

1. 修改 `onebot` 的配置文件，启用`正向ws`
1. 启动一次 `gypsum`，生成 `gypsum_config.toml` 配置文件
1. 向 `gypsum_config.toml` 配置文件中填写
1. 启动 `onebot` ，再启动 `gypsum`
1. 打开 `<你的ip地址>:9900`，开始使用
